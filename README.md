# Intoduction to Probability

This project aims to convert the book [Introduction to Probability](http://www.dartmouth.edu/%7Echance/teaching%5Faids/books%5Farticles/probability%5Fbook/book.html)
to [Mathbook XML](http://mathbook.pugetsound.edu/).

## Installation

For the conversion from latex to XML you need the install the python dependencies. You can do this with:

```
pip install -r requirements.txt
```
To compile a HTML version of the book you need to have `xsltproc` installed and the xslt stylesheets from the mathbook XML project available.
